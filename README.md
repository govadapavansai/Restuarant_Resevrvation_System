# Restaurant Reservation System

📌 Overview

This is a web-based restaurant reservation system that allows users to reserve tables online through a simple and user-friendly interface. Users can enter their reservation details, which are then stored in a MongoDB database using a Node.js and Express.js backend.

⚙️ Features

* Online table reservation form
* User-friendly interface
* Reservation data storage in MongoDB
* Backend API using Node.js and Express.js
* Real-time reservation submission and confirmation
* Input validation using HTML form validation

🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB
* Mongoose

🖥️ Usage

1. Start the MongoDB server.
2. Install required dependencies:
   npm install
3. Configure the MongoDB connection string.
4. Start the backend server:
   node server.js
5. Open index.html in a browser.
6. Click "Make a Reservation" and fill in the reservation details.

📋 Reservation Details

Users can provide:

* Name
* Email Address
* Number of Guests
* Reservation Date and Time

💾 Database Storage

All reservation details are stored in MongoDB using Mongoose, enabling efficient data management and retrieval.
